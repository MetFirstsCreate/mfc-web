# MFC Website

React-based website for Met Firsts Create (TMU Robotics Design Team).

## Quick Start

```bash
npm install
npm start
```

## Stack

-   React
-   Tailwind CSS
-   React Router

## Structure

```
src/
  components/     # Page components
  assets/        # Images & static files
```
